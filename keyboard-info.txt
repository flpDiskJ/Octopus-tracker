Main window (tracker):

    X CTRL+X = clear channel
    X CTRL+SHIFT+X = clear block
    X Delete & Backspace = delete step
    X ESC = toggle edit mode
    X Arrows = navigation

    skip:
        X CTRL+LEFT/RIGHT = change skip amount
        X tilde key = toggle skip between 1 and previously set amount
        X click on skip with mouse to toggle skip mode

    X TAB/SHIFT+TAB or ALT+LEFT/RIGHT = move cursor to prev/next channel
    X SHIFT+UP/DOWN = change block
    X SHIFT+LEFT/RIGHT = change instrument/sample
    X CTRL+N = append new block
    X CTRL+SHIFT+N = insert new block
    X CTRL+C/V = Copy/Paste channel
    X CTRL+SHIFT+C/V = Copy/Paste block
    X FUNC1-5 = change octave
    X FUNC6-10 or CTRL+1-5 = navigate block
    X SHIFT+ALT+0-9 Copy note to buffer
    X SHIFT+0-9 = Paste note from buffer

    X ALT+SPACE = play block
    X SHIFT+SPACE = play sequence
    X SPACE = stop all

      CTRL+O = open sequence order window
    X CTRL+H = open track properties window (bpm, speed, and such)
    X CTRL+B = open block properties window (block length, speed, and such)
      CTRL+F = open file op window (load/save module)
      CTRL+E = open edit window (copy buffers)
      CTRL+I = open instrument window
      CTRL+S = open sample edit window

    X CTRL+SHIFT+Q = quit

Block sub-window:
    X UP/DOWN = scroll block list
    X SHIFT+UP/DOWN = change block
    X click entry box with mouse to activate keyboard input
    X hit enter/return to exit keyboard input
